# Library Inventory Manager

A lightweight command-line Python application to manage books in a campus library.  
It allows library staff to track book status, search the catalog, and maintain records with persistent storage.

---

## Features

- Add new books to the library
- Issue books to users
- Return issued books
- View all books
- Search books by title or ISBN
- Persistent storage using `JSON`
- Robust file handling and logging
- Modular structure using Python packages
- Unit tests included
